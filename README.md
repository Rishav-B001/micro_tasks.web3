# micro_tasks.web3
# 🚀 Micro Tasks Platform (Rust + Stellar CLI)

A decentralized micro-task platform built with **Rust** and **Stellar Soroban (CLI)**. Users can create, accept, and complete small tasks with secure blockchain-based payments.

---

## 📌 Overview

This project enables:

- 📝 Task creation with rewards  
- 👨‍💻 Task participation and completion  
- 💸 Secure on-chain payments  
- 🔐 Transparent and trustless execution using smart contracts  

---

## 🛠️ Tech Stack

- **Rust**
- **Stellar Soroban**
- **Stellar CLI**

---

## ⚙️ Features

- ✅ Create micro-tasks  
- ✅ Accept tasks  
- ✅ Escrow-based payments  
- ✅ Verify task completion  
- ✅ Automatic reward distribution
